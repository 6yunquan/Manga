# Manga
**开源漫画阅读器，提供多个漫画来源。实现切换漫画来源、漫画推荐、漫画搜索、漫画收藏、漫画阅读、历史记录、缓存漫画
等功能。**

### 项目
* 项目基于RxJava+Retrofit2+Glide+ButterKnife，结合MVP模式开发。
* API由本人编写的Python项目提供，暂不提供源代码。
* 本项目纯属学习交流使用，不得用于商业用途。
